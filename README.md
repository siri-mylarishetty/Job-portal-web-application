# Mini Job Portal Web Application

A simple web-based Mini Job Portal developed using Flask and MySQL.  
This project allows employers to post job openings and applicants to register and apply for jobs.

## Features
- Applicant Registration
- Employer Registration
- Job Posting by Employers
- View Available Jobs
- Simple and User-Friendly Interface

## Technologies Used
- Frontend: HTML, CSS, Bootstrap
- Backend: Python (Flask)
- Database: MySQL (XAMPP)
- ORM: SQLAlchemy
- Version Control: Git & GitHub

## Project Structure
- app.py – Main Flask application
- template/ – HTML templates
- requirements.txt – Python dependencies
- .gitignore – Ignored files and folders

## How to Run the Project
1. Install Python
2. Install dependencies:
        pip install -r requirements.txt
3. Start MySQL server using XAMPP
4. Run the Flask app:

            flask --app app.py run

5. Open browser:

http://127.0.0.1:5000
